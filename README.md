## Overview

This is a frontend portfolio project simulating a social media user profile. Besides the UI itself, this repo shows how the project is structured, using Atomic Design Pattern.

## Build With

- NextJS (v13.2.4)
- React (v18.2.0)
- TailwindCSS (v3.3.1)
